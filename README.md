# Modern Blog Website

A simple and professional blog website built using HTML, CSS, and JavaScript.

The website displays blog post cards on the homepage. Each post includes an image, category, date, short summary, and a **Read More** button. When a user clicks a blog post, a new window opens with the full article content.

## Features

- Clean and professional homepage design
- Responsive layout for desktop, tablet, and mobile screens
- Blog cards with images, categories, dates, and summaries
- Full blog post opens in a new window
- 2-3 paragraph article content for each blog post
- Built with pure HTML, CSS, and JavaScript
- No external frameworks required
